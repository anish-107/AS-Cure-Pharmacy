# A S Cure Pharmacy

This repository contains the **complete source code and documentation** for the **A S Cure Pharmacy Website**.

The project is divided into separate modules for **frontend**, **backend**, and **documentation** to keep the codebase organized and maintainable.

---

## Repository Structure

```plaintext
A-S-Cure-Pharmacy/
│
├── frontend/        # React + Tailwind + TypeScript frontend application
│
├── backend/         # FastAPI backend service and API routes
│
├── docs/            # Project documentation and related resources
│
└── README.md        # Root repository documentation
````

---

## Development Overview

The application follows a **frontend-backend architecture**.

```plaintext
Frontend (React)
        │
        │ HTTP Requests
        ▼
Backend (FastAPI)
        │
        ▼
Services / Email / Database
```

The frontend communicates with the backend using **REST APIs**.
